# Daily Digest · 未来路线图

我们已经完成了资讯抓取、筛选、自动推送和内容池循环的核心链路，并实现了可视化的配置管理面板。接下来的目标是 **增强内容源**、**深化智能推荐** 并 **打通更多渠道**。

## ✅ 已完成功能

- [x] **可视化配置管理面板**
  - [x] 关键词配置（支持在面板中编辑）
  - [x] 调度配置（Cron 表达式或小时+分钟，数量控制）
  - [x] 企业微信模板配置（JSON 格式，支持占位符）
  - [x] 系统配置（管理员验证码、企业微信推送地址）
  - [x] 所有配置支持在线编辑和保存

---

### P0 · 核心功能增强

- [ ] **引入更多资讯源**
  - [ ] **RSS / Atom Feed**：支持批量添加 RSS 源，作为候选池的主要来源之一。
  - [ ] **GitHub Trending**：每日抓取 AI / Python / Engineering 等领域的 Trending 项目。
  - [ ] **Hacker News**：抓取高分（例如 >100 points）的 AI 相关主题。

- [ ] **打通微信公众号渠道**
  - [ ] 实现 `access_token` 的自动获取与刷新。
  - [ ] 实现“上传素材 → 创建草稿 → 发布图文”的核心 API 调用。
  - [ ] 在管理面板增加“一键同步到公众号”功能。

- [ ] **文章智能排序与推荐**
  - [ ] 为文章增加“热度分”或“推荐指数”，结合来源、时效性等因素。
  - [ ] 支持按“热度”或“时效”对候选池进行排序。
  - [ ] （可选）引入 Embedding，实现基于相似度的内容去重和主题聚类。

---

### P1 · 管理与易用性

- [ ] **多频道推送策略**
  - [ ] 支持配置多个企业微信机器人或公众号渠道。
  - [ ] 每个渠道可以关联不同的关键词、文章池和推送时间。

- [ ] **管理面板增强**
  - [x] **配置管理**：支持在面板中配置关键词、调度、模板、系统环境变量（已完成）
  - [ ] **批注与协作**：支持在面板上为文章添加内部评论或推荐语。
  - [ ] **批量操作**：支持在候选池中一键采纳/忽略某个关键词下的所有文章。
  - [ ] **数据统计**：简单统计每个关键词下的文章抓取量、采纳率等。

- [ ] **持久化方案升级**
  - [ ] 支持将 JSON 文件存储切换为 SQLite，提升数据操作的稳定性和性能。

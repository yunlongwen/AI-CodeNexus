## 100kwhy_wechat_mp

面向「AI 编程」的微信公众号后台与内容管线。

### 目标

- **公众号**：发布每日「AI 编程 & 团队管理日报」。
- **网站联动**：与站点 `https://www.100kwhy.fun/` 内容联动。
- **业务微信群**：将日报同步推送到业务微信群（优先考虑企业微信群机器人）。

### 技术栈（初版）

- 后端：Python 3 + FastAPI
- 任务调度：APScheduler（后续可接 crontab / 云原生方案）
- 运行环境：阿里云服务器（Linux）

### 模块规划

- `app/wechat`：微信公众号接入与消息处理
- `app/digest`：日报数据结构与生成流程
- `app/sources`：外部文章源（目标公众号、网站等）的采集与管理
- `app/notifier`：公众号群发接口封装 + 企业微信群机器人推送

后续会在此补充部署说明与开发指南。

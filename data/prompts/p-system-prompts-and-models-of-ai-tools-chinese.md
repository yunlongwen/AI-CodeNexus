# AI编程工具中文提示词合集

这是一个专门为中文开发者打造的AI编程工具提示词资源库，包含了多种主流AI编程工具的系统提示词和模型设计文档。

## 项目特色

- **全面覆盖**: 包含Cursor、Devin、VSCode Agent、Windsurf等多种AI编程工具
- **中文优化**: 所有提示词都经过中文环境优化，更适合中文开发者的使用习惯
- **持续更新**: 定期更新最新的AI编程提示词和中文编程Rules
- **实践导向**: 提供实际可用的提示词模板和最佳实践

## 包含工具

- Cursor Prompts - Cursor编辑器的提示词
- Devin AI - Devin AI的系统提示词
- VSCode Agent - VSCode Agent的相关文档
- Windsurf - Windsurf相关文档
- Claude Code - Claude Code提示词
- Trae - Trae AI工具提示词
- 以及更多AI编程工具

## 使用方法

1. 访问GitHub仓库: https://github.com/IsHexx/system-prompts-and-models-of-ai-tools-chinese
2. 根据您的开发环境选择对应的工具文件夹
3. 复制相应的提示词到您的AI编程工具中
4. 根据实际需求调整和优化提示词
